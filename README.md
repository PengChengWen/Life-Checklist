# 人生清单 - 微信小程序

一个美观实用的人生愿望清单小程序，帮助你记录这一生想要去做的事情。

## 功能特点

- ✅ **愿望管理**：添加、查看、完成、删除愿望
- 📊 **数据统计**：显示总愿望数、已完成数、未完成数
- 🔍 **筛选功能**：按全部/未完成/已完成筛选愿望
- 🎨 **美观UI**：现代化的渐变设计，流畅的动画效果
- 💾 **本地存储**：使用微信小程序本地存储，数据安全可靠
- ⭐ **优先级标记**：支持高/中/低三个优先级
- 📝 **默认数据**：内置8条精美的默认愿望示例

## 使用说明

### 1. 开发环境

- 微信开发者工具（最新版本）
- 小程序AppID（可在微信公众平台申请）

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"小程序项目"
3. 导入本项目目录
4. 输入你的AppID（测试可以使用测试号）

### 3. 图标配置（可选）

**方案一：使用TabBar（推荐）**

如果想使用底部TabBar导航：

1. 创建 `images` 目录
2. 添加以下图标文件（每个图标尺寸为 81x81px）：
   - `list.png` - 列表图标（未选中）
   - `list-active.png` - 列表图标（选中状态，紫色 #6366f1）
   - `add.png` - 添加图标（未选中）
   - `add-active.png` - 添加图标（选中状态，紫色 #6366f1）

你可以使用在线图标生成工具或图标库（如 iconfont）创建这些图标。

**方案二：不使用TabBar**

如果暂时没有图标，可以使用备选配置：

1. 将 `app-no-tabbar.json` 的内容复制到 `app.json`
2. 或直接删除 `app.json` 中的 `tabBar` 配置
3. 使用页面右上角的浮动"+"按钮或通过导航栏跳转到添加页面

**注意**：本项目已添加浮动添加按钮，即使没有TabBar也能正常使用。

### 4. 功能说明

#### 清单列表页面
- 查看所有愿望清单
- 点击筛选标签切换全部/未完成/已完成
- 点击左侧圆圈标记完成/取消完成
- 点击"删除"按钮删除愿望
- 查看统计数据和优先级标记

#### 添加愿望页面
- 输入愿望标题（必填）
- 输入愿望描述（必填）
- 选择优先级（低/中/高）
- 点击"添加愿望"保存

## 技术实现

### 数据存储
使用微信小程序的 `wx.setStorageSync` 和 `wx.getStorageSync` API 进行本地数据存储。

### 核心功能
- **数据管理**：完整的CRUD操作
- **状态管理**：愿望完成状态切换
- **时间格式化**：友好的相对时间显示
- **数据排序**：按状态、优先级、创建时间智能排序

### UI设计
- 渐变色背景（紫色系）
- 卡片式布局
- 圆角和阴影效果
- 流畅的交互动画
- 响应式设计
